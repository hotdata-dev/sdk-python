# ResultsFormatQuery

The `?format=` query parameter on `GET /v1/results/{id}`. One of `arrow`, `json`, `csv`, `md`, or `parquet`.

## Enum

* `ARROW` (value: `'arrow'`)

* `JSON` (value: `'json'`)

* `CSV` (value: `'csv'`)

* `MD` (value: `'md'`)

* `PARQUET` (value: `'parquet'`)
